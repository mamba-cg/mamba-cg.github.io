FAQ

Common questions.

Short buyer-focused answers for installation, collections, foam, ripples, floating objects and presets.

How do I start?

Install and enable the addon, open the 3D View sidebar, then click Create Stylized Water.

What does the button create?

It creates or reuses SWP_Water, assigns the water material, adds Geometry Nodes modifiers and creates required collections.

How do I create foam around objects?

Select your object and use Add Selected to Static so it becomes part of the Static_Objects workflow.

How do I create ripples?

Add selected objects or proxies to Ripple_Points and adjust the ripple controls in the panel.

How do floating objects work?

Add selected objects to Floating_Objects so the addon can set up floating-related controllers and constraints.

Can I use presets?

Yes. Use built-in presets or save custom presets as JSON files in the addon presets folder.

What if the asset file is missing?

Make sure Water_Surface_MambaCG.blend exists inside the addon assets folder.

Are the nodes editable?

The system uses Blender materials and Geometry Nodes, so advanced users can inspect and customize the setup.
