Stylized Water Pro page - iframe integration

Folder structure:
- index.html
- css/styles.css
- js/main.js
- assets/

How to use:
1. Upload this folder to your hosting or GitHub Pages.
2. Replace the placeholder media blocks in index.html with real images, videos or GIFs.
3. Paste the iframe into the SuperHive editor.

Example iframe:
<iframe src="https://your-domain.com/stylized-water-pro-page/index.html" width="100%" height="7200" style="border:0; width:100%; max-width:100%;" loading="lazy"></iframe>

Recommended iframe height:
- Desktop: 7000-7600 px
- Mobile: 9000-10000 px

Recommended media:
- hero looping GIF or MP4: final water render with waves, foam, ripples and floating object
- overview video: empty scene to final water system
- UI screenshot: 3D View sidebar panel
- foam/ripple GIF: object interaction
- floating object GIF: floating behavior
- before/after render: same scene before and after Stylized Water Pro
